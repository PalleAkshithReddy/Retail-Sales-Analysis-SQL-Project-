# Retail-Sales-Analysis-SQL-Project-

This project focuses on designing a simple online retail sales database to organize and analyze customer, product, and order data. Using a structured relational model with tables for Customers, Products, Orders, and Order_Items, it ensures data consistency and clear relationships. With SQL queries, the database helps identify top-selling products, high-value customers, revenue trends, and inactive users. Overall, it demonstrates how raw sales data can be transformed into meaningful insights to support better business decisions.
